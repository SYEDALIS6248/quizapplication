📝 Interactive Quiz Application

A full-stack quiz application featuring secure user authentication, role-based dashboards (Admin & Student), and dynamic quiz management.

🚀 Features

Role-Based Access Control: Separate dashboards and functionalities for Administrators and Students.

Admin Dashboard: Create new quizzes, add multiple-choice questions, and manage existing content.

Student Dashboard: View available quizzes, take timed tests, and view past attempts.

Secure Authentication: User login and registration system with password validation.

Global Leaderboard: Compare your scores with other students.

Modern UI: Responsive design built with modern CSS properties, custom variables, and FontAwesome icons.

🛠️ Technologies Used

Frontend: HTML5, Modern CSS (Flexbox/Grid, Custom Properties), Vanilla JavaScript

Backend Integration: REST API calls via JavaScript Fetch API

Version Control: Git & GitHub

📂 Project Structure

quizapplication/
├── html/          # HTML pages (index.html, admin.html, student.html, quiz.html)
├── css/           # Stylesheets (global.css, admin.css, quiz.css)
└── js/            # JavaScript logic (auth.js, admin.js, student.js, api.js)


💻 How to Run Locally

git clone [https://github.com/SYEDALIS6248/quizapplication.git](https://github.com/SYEDALIS6248/quizapplication.git)
